Boilerplate for maven project with groovy support.

What you get what is supported:
- java 1.7
- groovy 2.2.2 (with static compilation, type check etc.)
- groovy-eclipse-compiler plugin used to compile groovy sources
- gpars 1.1.0
- logback (logging)
- utf-8 encoding

Test dependencies:
- fest assertions
- testng
- mockito
